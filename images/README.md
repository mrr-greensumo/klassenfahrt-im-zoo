# Bilder für das Geocache-Rätsel

Platziere hier deine Bilder:

## Erforderliche Dateien

### Bus-Bild
- **Dateiname:** `bus.png`
- **Format:** PNG mit Alpha-Kanal (Transparenz)
- **Auflösung:** Empfohlen 1600x900 px (16:9)
- **Inhalt:** Bus mit 8 transparenten Fenstern, gleichmäßig verteilt

### Tier-Bilder (8 Stück)
- **Dateinamen:**
  - `tier1.png`
  - `tier2.png`
  - `tier3.png`
  - `tier4.png`
  - `tier5.png`
  - `tier6.png`
  - `tier7.png`
  - `tier8.png`
- **Format:** PNG mit transparentem Hintergrund
- **Größe:** Empfohlen 200x200 px (werden automatisch skaliert)

## Tipps

1. **Bus-Fenster:** Achte darauf, dass die Fenster groß genug sind, damit die Tiere gut sichtbar sind
2. **Tiere:** Wähle Tiere mit klaren Silhouetten (gut erkennbar)
3. **Transparenz:** Stelle sicher, dass der Hintergrund wirklich transparent ist (kein weißer Hintergrund)
4. **Kontrast:** Die Tiere sollten sich gut vom Hintergrund abheben

## Beispiel-Tool zum Erstellen

- **GIMP** (kostenlos) - für Bus mit transparenten Fenstern
- **Photoshop** - professionelle Lösung
- **Online:** [remove.bg](https://remove.bg) - zum Entfernen von Hintergründen
- **Canva** - für einfache Grafiken
